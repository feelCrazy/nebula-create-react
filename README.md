### 安装

```bash
npm install nebula-create-react -g
# or
yarn global add nebula-create-react
```

### 如何使用

```
> mkdir  [projectName]
> cd  [projectName]
> create-react
```

#### **生成项目文件结构**

```
|-- /src
    |-- /api
        |-- ...
    |-- /components
        |-- /...
            |-- ...
            |-- ...
    |-- /container
        |--/...
    |-- /res
    |-- /models
        |-- ...
    |-- /router
        |-- ...

```
